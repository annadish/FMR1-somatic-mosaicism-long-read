# FMR1-somatic-mosaicism-long-read
Analysis scripts supporting the manuscript: “Long-read sequencing reveals extensive FMR1 somatic mosaicism in FXTAS human brain.”

## Overview
This repository contains R scripts used to analyze amplification-free long-read sequencing data to quantify:
  - FMR1 CGG repeat length distributions
  - Somatic instability
  - Per-read CpG methylation fractions
  - Repeat length–methylation coupling
  - Cohort-level statistical analyses

## Data Availability
Processed summary data are available in the supplemental materials of the manuscript.

## Analyses
All analysis was performed in R.
